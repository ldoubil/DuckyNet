# DuckyNet Web Admin - 快速开始

## 🚀 第一次启动

### 1. 安装 Node.js
如果还没安装Node.js，请访问：https://nodejs.org/

### 2. 安装依赖
```bash
cd WebAdmin
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问界面
打开浏览器访问：http://localhost:3000

## ⚡ 后续启动

### 方式1：使用启动脚本（推荐）
双击 `启动前端.bat`

### 方式2：命令行
```bash
cd WebAdmin
npm run dev
```

## 🔌 连接后端

确保后端 Server 正在运行：
- API 地址: http://localhost:5000
- WebSocket: ws://localhost:5000/ws

如果后端端口不是5000，请修改 `.env` 文件

## 📝 开发模式

- 自动热重载（修改代码立即生效）
- Vite 代理（自动转发API请求）
- DevTools 支持

## 🏗️ 生产构建

```bash
npm run build
```

生成的文件在 `dist/` 目录

## ❓ 遇到问题？

### 端口被占用
修改 `vite.config.js` 中的端口

### 依赖安装失败
```bash
rm -rf node_modules
npm install
```

### 无法连接后端
检查后端是否运行在 http://localhost:5000
